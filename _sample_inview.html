inview 範例

可透過 js 資料夾的 jquery.inview_set.js 或 css 資料夾的 inview.css 進行修改或新增。

若新建頁面也想使用動畫效果，請將本頁面 HTML 下方的
<!--視差效果 (inview)-->
<script src="https://cdnjs.cloudflare.com/ajax/libs/protonet-jquery.inview/1.1.2/jquery.inview.min.js"></script>
<script src="js/jquery.inview_set.js"></script>
複製貼上至相同位置。若無這些標籤，動畫將無法運作。
此外，若長期使用，inview 版本過舊可能導致無法運作。屆時請搜尋「CDN inview」等重新套用。
